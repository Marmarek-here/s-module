# Now on GitHub!

Just a little small module made to display Python's rudimentary legs with any length (ironically, the library is for Python). Yes, it’s exactly as unnecessary as it sounds.
Usage:  
```Python
import s    
leg = s(X) # X is the amount of = you want in the leg    
print(leg)  
```
